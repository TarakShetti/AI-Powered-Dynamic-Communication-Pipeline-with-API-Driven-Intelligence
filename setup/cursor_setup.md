
# ⚙️ Cursor IDE Setup with MCP

## Overview

This document explains how MCP servers are integrated into Cursor AI.

---

## Setup Steps

1. Install Cursor AI Editor
2. Go to Settings → Tools & Integrations
3. Add Custom MCP Server
4. Paste MCP configuration

---

## Example Config

```json
{
  "Gmail": {
    "url": "https://mcp.pipedream.net/your-endpoint/gmail"
  }
}
```

---

## Notes

* Keep MCP URLs private
* Authentication is handled externally
